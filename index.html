③ 提案 — あなたのリグ向け（HX Stomp / ブースター / IR）
START を押してギターを弾いてください。
使い方: START → 目標を選ぶ/入力 → 数秒間しっかり弾く（リフでもソロでも）。鳴っている音を解析して、目標との差を具体的なツマミ操作に翻訳します。 精度のコツ: マイク/部屋/IRの色付けで絶対値はブレます。まず今の音を「基準」に保存し、ツマミを変えながら差分で追うと信頼度が上がります。提案は方向のヒント（"Midを上げる"等）であって絶対値ではありません。 解析は HX Stomp などを通した処理後の音（インターフェイス入力）を対象にすると精度が出ます。
